# Генератор надёжных паролей с оценкой стойкости

Веб-приложение для генерации случайных паролей с возможностью настройки параметров и проверки их сложности.  
Работает локально, не требует интернета. 
---

## Возможности

- Настройка длины пароля (от 4 до 32 символов)
- Выбор наборов символов: цифры, буквы (верхний/нижний регистр), спецсимволы
- Генерация случайного пароля с использованием `secrets` (криптографически стойкий)
- Оценка стойкости пароля: слабый / средний / сильный / очень сильный
- Копирование пароля в буфер обмена (через JavaScript)
- История последних 5 сгенерированных паролей (сохраняется в сессии)

## Технологии

- **Backend:** Python 3, Flask
- **Frontend:** HTML5, CSS3, JavaScript
- **Дополнительно:** HTML-шаблоны Jinja2, сессии Flask

## Установка и запуск локально

### Требования
- Python 3.8 или выше
- pip

### Шаги

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/artyushina/password-generator.git
   cd password-generator
   ```

2. **Создайте и активируйте виртуальное окружение** 
   ```bash
   python -m venv venv
   source venv/bin/activate    # для Linux/Mac
   venv\Scripts\activate       # для Windows
   ```

3. **Установите зависимости**
   ```bash
   pip install flask
   ```

4. **Запустите приложение**
   ```bash
   python app.py
   ```

5. **Откройте в браузере**
   Перейдите по адресу [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Использование

1. На главной странице выберите параметры пароля:
   - Длина (слайдер)
   - Какие символы использовать (чекбоксы)
2. Нажмите **«Сгенерировать»**.
3. Получите пароль и его оценку сложности (цветной индикатор).
4. При необходимости нажмите **«Копировать»**, чтобы сохранить пароль в буфер обмена.
5. История последних паролей отображается внизу страницы.

## Алгоритм оценки сложности

Оценка основана на длине пароля и разнообразии использованных символов:

- **Очень слабый** (< 8 символов)
- **Слабый** (8–10 символов, только один тип символов)
- **Средний** (8–10 символов с несколькими типами ИЛИ 11–14 символов с одним типом)
- **Сильный** (11–14 символов с несколькими типами ИЛИ 15+ с одним типом)
- **Очень сильный** (15+ символов с несколькими типами)

Также учитывается, были ли использованы цифры, буквы разного регистра, спецсимволы.
